BioPath Pitch One-page Guide

Goal: finish one full demo in 2 minutes and deliver a clean Problem → Solution → Proof → Ask story.

Back to Pitch UI

30-second startup

  1. Open BioPath UI and click Auto Connect.
  2. If needed, paste API URL and click Health Check.
  3. Keep default Cambridge demo map.
  4. Check top status is Connected.

2-minute demo script

  1. Click Run Solve (trap coordinates generated).
  2. Click Run Benchmark (proof versus random baseline).
  3. Read out Capture Probability and Uplift.
  4. Click Open Pitch Mode.
  5. Use copy buttons for Problem / Solution / Proof / Ask.

Failure recovery (现场救场)

  • API fail: click Fix in 1 step, then Health Check.
  • Map JSON error: read line/column hint under map and fix that line only.
  • Benchmark timeout: click Retry with lower MC.
  • No runs shown: click Run first benchmark now.

60-second pitch lines

Problem: Manual trap placement is inconsistent and misses bottlenecks in complex farm layouts.

Solution: BioPath turns a simple map into optimized trap coordinates and a reproducible plan.

Proof: We benchmark against random baseline with Monte Carlo validation and robust capture metrics.

Ask: 2 pilot introductions, small PoC budget, and practical data access for an 8-week farm trial.
