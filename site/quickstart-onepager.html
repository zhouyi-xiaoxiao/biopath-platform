30-second startup

  1. Run bash scripts/start_public_demo.sh on your laptop.
  2. Open Pitch Studio and click Auto Connect.
  3. Confirm status shows Connected (or Pitch Safe Ready if offline).
  4. Keep default Cambridge map and k = 6.

2-minute demo flow

  1. Click Run Solve.
  2. Click Run Benchmark.
  3. Read five values: optimized capture, heuristic baseline, uplift, robust score, MC runs.
  4. Switch to pitch-safe narrative view and deliver ask.

Failure recovery

  • API unreachable: stay in Pitch Safe mode and continue script.
  • JSON map error: fix line/column shown under map box.
  • Benchmark timeout: lower MC runs and retry.
  • No run list: read curated proof from Proof Panel.
