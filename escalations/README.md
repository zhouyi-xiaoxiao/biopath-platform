# Escalations

- `pending/*.md`: difficult problems queued for GPT-5.2 Pro extended-thinking web escalation.
- `answers/*.md`: captured assistant responses from chatgpt.com.
- `actions/*.md`: Codex implementation reports based on answers.

## Pending file format

```md
# Problem
Explain the blocking issue.

## Context
- Relevant files
- Current behavior
- Expected behavior

## Constraints
- Tests to keep green
- Performance limits
```
